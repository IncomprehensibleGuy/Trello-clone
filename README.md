# Project setup:
* `git clone https://github.com/IncomprehensibleGuy/trello-clone.git`
* `cd trello-clone`
* `npm i`
* `python -m venv venv`
* `cd venv/Scripts` => `activate` => `cd ../..`
* `pip install -r requirements.txt`
* `manage.py migrate`
* `manage.py runserver`
* in new terminal `npm run watch`
